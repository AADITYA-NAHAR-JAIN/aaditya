A modern, interactive personal portfolio website showcasing social media links, professional achievements, and creative work. Built with cutting-edge web technologies featuring animated particle systems, 3D card effects, and smooth scroll animations.

## 🌟 Features

- **Interactive Particle System**: Dynamic background with connecting particles that respond to mouse movement
- **3D Card Effects**: Hover animations with tilt and glow effects on social media cards
- **Custom Cursor**: Glowing cursor that follows mouse movement
- **Smooth Animations**: Staggered loading animations and parallax scrolling effects
- **Responsive Design**: Optimized for all device sizes
- **Performance Optimized**: Lazy loading and reduced motion support
- **Social Integration**: Direct links to all professional and personal profiles

## 🚀 Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced animations, gradients, and responsive design
- **JavaScript (ES6+)**: Interactive features and animations
- **Font Awesome**: Icon library for social media icons
- **Google Fonts**: Custom typography (Orbitron & Syne)

## 🛠️ Setup & Installation

1. **Clone or Download** the repository
2. **Open** `index.html` in your web browser
3. **Enjoy** the interactive experience!

No additional dependencies or build tools required - it's a pure static website.
